widget_tool
===============================

A Custom Jupyter Widget Library

Installation
------------

To install use pip:

    $ pip install widget_tool
    $ jupyter nbextension enable --py --sys-prefix widget_tool


For a development installation (requires npm),

    $ git clone https://github.com//widget_tool.git
    $ cd widget_tool
    $ pip install -e .
    $ jupyter nbextension install --py --symlink --sys-prefix widget_tool
    $ jupyter nbextension enable --py --sys-prefix widget_tool
